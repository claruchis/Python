# Proyecto Final Python - Vivero 🌻

Aplicación desarrollada como proyecto final del curso de Python.

Se trata de un sistema de gestión de inventario para un vivero, ejecutado desde consola.

## 🔧 Tecnologías utilizadas

* Python

## ⚙️ Funcionalidades

* Agregar productos al inventario
* Ver listado de productos
* Buscar productos por nombre
* Modificar stock
* Actualizar precio
* Eliminar productos
* Reporte de productos con bajo stock

## ▶️ Cómo ejecutar

1. Clonar el repositorio
2. Ubicarse en la carpeta del proyecto:

cd proyecto_vivero

3. Ejecutar:

python3 main.py

## 🧠 Conceptos aplicados

* Funciones
* Condicionales
* Bucles
* Manejo de listas y diccionarios
* Validación de datos con try/except
* Interacción con el usuario (input)
